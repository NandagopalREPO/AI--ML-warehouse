What is Machine Learning?

Machine Learning is a field of AI where systems learn patterns from data and make predictions or decisions without being explicitly programmed.

Instead of writing rules:

Traditional programming → Rules + Data → Output
Machine Learning → Data + Output → Model (learns rules)
Types of Machine Learning
1. Supervised Learning
Model learns from labeled data
Input → Output mapping is known

Examples:

House price prediction
Spam detection

Common Algorithms:

Linear Regression
Logistic Regression
Decision Trees
Random Forest
2. Unsupervised Learning
No labels, model finds hidden patterns

Examples:

Customer segmentation
Anomaly detection

Algorithms:

K-Means Clustering
Hierarchical Clustering
PCA
3. Reinforcement Learning
Agent learns by trial and error

Examples:

Self-driving cars
Game playing (AlphaGo)

Key Concepts
1. Features & Target
Features (X): Input variables
Target (y): Output variable
2. Training vs Testing
Train → Learn patterns
Test → Check performance
3. Overfitting vs Underfitting
Overfitting: Model memorizes data (bad generalization)
Underfitting: Model too simple

Goal: Balance both

Bias vs Variance
Bias: Error due to wrong assumptions
Variance: Sensitivity to data changes

Trade-off is key in ML

Evaluation Metrics

When to Use Machine Learning?

Use ML when:

Rules are complex or unknown
Large amount of data is available
Patterns exist but are hard to code
When NOT to Use ML
Simple rule-based problems
Very small datasets
When interpretability is critical (sometimes)
Real-World Applications
Fraud detection
Recommendation systems
Predictive maintenance
Healthcare diagnosis
